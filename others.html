HTMLについて（簡単なページを作ってみる）

upload:2014.02.09

HTMLは、Hyper Text Markup Language の略で、
Webページ（ホームページ）を書くための基本中の基本。

ハイパーテキストマークアップランゲージ???

何だか難しそうですが、実はそうでもありません。
これがないと何も始まらないのですが、　
逆に言うと、シンプルな WebページならHTMLだけで作れます。

「ホームページビルダとかのソフトがいるんでしょ？」
などど、僕も最初は思っていました。

いざ始めるまでは敷居が高く見えるのですが
実は。。。

テキストエディタ（Windowsならメモ帳）だけで作れます！！！

物は試しです、色々と考えるよりやってみた方が分かりますよね。
下にとても短いHTMLを用意しましたので、気軽な気持ちで作ってみましょう！ 

                <!DOCTYPE html>
                <html lang="ja">
                    <head>
                        <meta charset="utf-8">
                        <title>最初の一歩！！！</title></title>
                        <meta name="description" content="とにかく作ってみる♪">    
                    </head>
                    <body>
                        <h1>おめでとう</h1>
                        <p>初めてのWebページができましたね♪</p>
                    </body>
                </html>
  1. テキストエディタ（メモ帳など）を起動
  2. 上の文字列をコピーして、テキストエディタ（メモ帳など）にペースト
  3. ペーストしたファイルに index.html と名前を付け、保存
    （名前は半角英数字ならば自由に付けても構いませんが、拡張子は.htmlにしましょう）
  4. 保存したファイル、index.html をダブルクリックして開く
    （もしくは、index.htmlをブラウザにドロップ）

どうでしょうか?

うまく作成されていれば、左の画像のようにシンプルなWebページが表示されているはずです。
このシンプルなWebページに色々な仕掛けを追記していくことで、みなさんが普段よく見ている色鮮やかで、使いやすいWebページになっていきますよ。

この調子で、勉強を進めていきましょう！
