Gitについて

upload:2014.03.30

Gitってご存知ですか?
「ジット」ではなく「ギット」と読みます。

GitHub ←GitHub（ギットハブ）の公式HPはこちらです

Gitは一言でいうと、
作成しているプログラムなどのバージョンを管理する仕組みです。　

難しい言葉では、VCS:Version Control System 日本語では、
バージョン管理システム（難しくなかったですね....）
と呼ばれるものです。

プログラムなどのバージョン管理と言いましたが、
別にプログラムじゃなくても構いません。

たとえばPCで絵を描く人なら、その製作過程を記録しておくことができます。
PCで絵を描く過程が、

下絵　→　下絵の修正　→　着色1　→　着色2　→　微調整　→　完成

だとしましょう。

この「下絵」や「着色2」など、それぞれの過程におけるデータを
Gitでは保存しておくことができます。
（実際には全部保存している訳ではないのですが、そのように扱うことができます）

しかもデータを保存したフォルダをGitに登録しておけば
半自動的に変更履歴を保存してくれます。
（なぜ「半」自動的なのかは次回の投稿で記載しますね）

ですから、手元のデータがたとえ「微調整」の段階に入っていても、
「下絵」や「着色1」など
データを以前の状態まで戻すことができるのです。

しかも、いつ、誰が、何を、どのように変更したのかが確認できるのです。

ここで「誰が」という言葉が出てきて気づいた方もいらっしゃるでしょう。
そう、Gitでは
複数の人でひとつの作品を作ることも可能なのです。

ということは、もちろん

ファイルの共有も可能です。

かなり使い勝手の良いシステムなので世界中で多くの人々が使っています。
実際、僕もこのHPのデータ管理をGitで行っています。

素晴らしいでしょ、Git。

あなたも使ってみませんか?

近日中にGitの導入法とその簡単な使い方について記事を投稿しますのでお楽しみに♪

Bootstrapメモ

upload:2014.10.21

Bootstrap（ブートストラップ）は今風のWebサイトを簡単に作るための
CSSフレームワークです。

Bootstrap ←Bootstrap（ブートストラップ）の公式HPはこちらです

CSSフレームワークというのは、
ひとつひとつ手作業でCSSを記述する手間を省くために用意されている　

「魔法の呪文集」みたいなものです。

このBootstrapの導入方法や記述の仕方について、
ここにメモを残していきたいと思います。

導入の方法

まずは Bootstrapの公式ページから必要なファイルをダウンロードします。

"Download Bootstrap"のボタンをクリックします。

Downloadページに移動します。
一番左の"Download Bootstrap"のボタンをクリックすると、
ダウンロードが始まります。
ダウンロードされたzipファイルを解凍してください。
bootstrap-x.x.x（xは数字です）という名前のフォルダが出来ています。

このbootstrap-x.x.x（xは数字です）フォルダ内に、
htmlファイルを作成します。
例えば、index.htmlという名前のファイルを作成し、
このindex.html内に、
Downloadページ内の"Basic Template"を全てコピーしてペーストします。

実は。。。。
これだけで準備完了！なのです。
どうでしょうか？簡単ですね。

準備ができたので、次はその使い方について説明します。

グリッドレイアウトを使ってみる

それでは早速、BootstrapでWebページを作成してみましょう。
さきほど作ったindex.htmlのbodyタグ内に以下のコードを記述します。 
                <div class="container">
                      <div class="row">
                        <div class="col-sm-2" style="background:gray;">2Columns</div>
                        <div class="col-sm-6" style="background:lime;">6Columns</div>
                        <div class="col-sm-4" style="background:orange;">4Columns</div>
                      <div>
                 <div>

変更したindex.htmlをブラウザで開くと、



こんな感じで分割されているのが分かると思います。
上の画像では、わかりやすいように1列ずつのブロックも表示してありますが、
さきほど記述したのは下段のグレー、緑、オレンジの部分だけです。

このように、
BootstrapではWebページを12列に縦割り分割したグリッドシステムが使えます。

このグリッドシステムを使うには、
まず対象範囲に"row"というclass指定を行います。
<div class="row">...<div>の部分がそれにあたります。

そして、例えば2列,6列,4列に分割する場合には、それぞれ

<div class="col-sm-2" >...</div>
<div class="col-sm-6" >...</div>
<div class="col-sm-4" >...</div>

と指定します。

ここで大切なことは、全体を12列に分割しているので、
col-sm-に続く数字の合計が12
になるようにしなければいけない、ということです。

この数字の合計が12ではない場合も機能はしますが、
レイアウトが崩れてしまうので注意が必要です。
